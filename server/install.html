Install – Click the button or go to the Chrome Web Store.

If you use Google Chrome (stable) on Windows, this is the only option.

Install MLE in Chrome

Install – Drag&drop the mle.crx file on the chrome://extensions page. (Screenshot.)

If you use Chromium or Chrome (dev) or Chrome not on Windows, you can install it from here without using the Web Store.

Install MLE in Firefox

Install – Click the button.

Install MLE in Opera 12

Install – You have to add this domain (https://sebadorn.de/) to the list of trusted websites. (Screenshot.) Then click the button.

Install MLE in Opera 15+

Install – Drag&drop the mle.crx file into Opera. You may have to enable it afterwards in the Opera extensions overview.

Alternative – Use this extension and install MLE from the Chrome Web Store.
